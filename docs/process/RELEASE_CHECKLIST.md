# 发布与演示检查清单

## 产品范围

- [ ] P0 用户故事全部完成或有明确例外说明。
- [ ] 未完成的 P1/P2 已移入后续 Backlog。
- [ ] 演示流程和测试账号已准备。

## 功能验证

- [ ] Cognito 登录、退出和 Token 过期处理正常。
- [ ] 图片上传、处理、缩略图和标签正常。
- [ ] 视频上传、抽帧、聚合和结果显示正常。
- [ ] My Media、Shared Media 和 Results 状态正确。
- [ ] 标签、文件和缩略图查询正常。
- [ ] 手动标签与批量操作正常。
- [ ] 所有者删除正常，非所有者操作被拒绝。
- [ ] 重复上传检测正常。
- [ ] SNS 订阅、确认、触发和取消正常。

## 安全与数据

- [ ] 仓库中没有 `.env`、Token、密码、私钥或云访问密钥。
- [ ] 本地数据库、原始媒体和临时查询文件未提交。
- [ ] 后端不信任浏览器提交的 owner 信息。
- [ ] 原始媒体和缩略图不存在非预期公开访问。
- [ ] 日志不包含 JWT 或其他敏感信息。

## 可靠性

- [ ] SQS 失败重试行为已验证。
- [ ] 重复消息不会创建重复记录。
- [ ] Cloud Run 超时和异常有可诊断日志。
- [ ] 关键资源的超时、并发和限制配置已检查。

## 测试

- [ ] 后端测试通过。
- [ ] 集成测试通过。
- [ ] 权限与跨用户测试通过。
- [ ] 至少一个图片和一个短视频完成端到端验证。
- [ ] 主要桌面浏览器完成 Smoke Test。

## 文档与作品集证据

- [ ] README 中的运行和部署信息与当前版本一致。
- [ ] 架构图与实际实现一致。
- [ ] PRD、验收标准、Sprint 计划和决策记录已归档。
- [ ] 个人贡献描述可由界面、提交或报告支持。
- [ ] 所有量化结果附带测量口径；没有证据的数字已删除或降级表述。
- [ ] 明确仓库 UI 仅用于 Demo 和功能测试，不代表最终客户 UI。

## 客户与 UI 供应商交接

- [ ] 客户已收到约定的部署、测试和已知限制说明。
- [ ] UI 外包团队已收到 API Contract 和调用示例。
- [ ] Cognito 登录和 Token 传递方式已说明。
- [ ] pending、processing、completed 和 failed 等状态已说明。
- [ ] My Media、Shared Media、标签、删除和通知权限边界已说明。
- [ ] 已建立交付后的问题反馈和售后响应渠道。

## 演示前

- [ ] AWS 与 GCP 服务健康。
- [ ] Cloud Run 已预热。
- [ ] 准备一个小图片和一个短视频。
- [ ] 准备已处理完成的备用媒体。
- [ ] 邮件订阅已完成确认。
- [ ] 准备架构图、备用截图或录屏。
